# Philamentix Hub V17.2.1

## Bambu Studio Bridge

- STL- und 3MF-Dateien direkt aus der Druckbibliothek in Bambu Studio öffnen
- Windows-Protokollhandler `philamentix-bambu://`
- geschützte Supabase-Dateien werden über eine kurzlebige Signed URL geladen
- Bestätigungsdialog vor jedem Download
- nur HTTPS sowie STL/3MF zugelassen
- normaler Download bleibt als Fallback verfügbar

## Einrichtung

In der Druckbibliothek oben **Bambu-Bridge** auswählen, ZIP entpacken und `Philamentix-Bambu-Bridge-installieren.bat` ausführen.
